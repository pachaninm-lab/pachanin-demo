# Прототип «детски простого» ЛК

`lk-prototype.html` — самодостаточный кликабельный прототип кабинета покупателя
(открывается любым браузером, без сборки). Демонстрирует целевой уровень простоты
из `DESIGN_PLAYBOOK_cabinets_2026-07-16.md`.

## Шесть правил «понятно ребёнку», применённых в прототипе

1. **Одно дело за раз.** Первый экран начинается с приветствия и одной фразы:
   «Сегодня одно важное дело и два обычных». Дело-герой — одна карточка с одной кнопкой.
2. **Язык десятилетнего.** «Банк должен подтвердить, что деньги готовы» вместо
   «резерв ожидает банковского подтверждения». Каждый статус объясняет последствие:
   «Пока банк не ответит, зерно не поедет к вам».
3. **Светофор вместо статусной системы.** Красная точка — нужно ваше действие,
   жёлтая — ждём кого-то другого, зелёная — готово, серая — не срочно.
4. **Деньги — тремя строками.** Свободны / Отложены / На паузе, у каждой строки
   одна причина и одна ссылка «Почему?».
5. **Сделка — как история.** Таймлайн простыми словами («Машину взвесили»,
   «Вы договорились о цене»), сверху самое новое. Профессиональные детали —
   в свёрнутом блоке «Подробности для бухгалтера и юриста».
6. **Действие = обещание.** Экран решения говорит, что произойдёт (3 шага),
   что не произойдёт («деньги никуда не спишутся») и подтверждает результат
   экраном «Запрос отправлен!» с инструкцией «можно закрыть страницу».

Сценарий для проверки: Главная → «Попросить банк» → «Отправить запрос банку» →
успех → главная переключается в состояние «Ждём банк». Тёмная тема — автоматически
от системной настройки.
