# 🛡️ Свой VPN навсегда: Oracle Cloud + VLESS-Reality

Это план, который выбрал консилиум. Результат: **личный сервер, бесплатный навсегда**
(4 ядра / 24 ГБ / безлимит трафика), протокол **Reality** — почти не блокируется в РФ,
работает на iPhone и Samsung через Happ.

Твоё участие — один раз ~15 минут. Всё сложное делает скрипт `setup-reality.sh`.

---

## Почему именно так (коротко)

- РКН с февраля 2026 целенаправленно убивает **VLESS+WebSocket+TLS** — это причина, по которой
  все публичные конфиги постоянно дохнут. Лечению ротацией не поддаётся.
- **Reality** маскирует трафик под настоящее TLS-рукопожатие к реальному сайту
  (`www.microsoft.com`). При активной проверке РКН видит обычный microsoft.com, а не прокси.
  Сертификата-подделки нет → ловить нечего. Обход ~98–99%.
- Reality работает **только на настоящем сервере** (Дено/Cloudflare/Vercel так не умеют).
  Самый дешёвый «настоящий сервер навсегда» = **Oracle Cloud Always Free**.

---

## ЧАСТЬ 1. Создать аккаунт Oracle Cloud (один раз, ~10 мин)

1. Открой **https://www.oracle.com/cloud/free/** → кнопка **Start for free**.
2. Заполни регистрацию:
   - **Country/Territory:** Россию выбрать нельзя — выбери любую доступную (например
     **Germany / Netherlands / Kazakhstan**). На работу VPN это не влияет.
   - Email, имя — реальные (на email придёт подтверждение).
3. **Home Region** — выбирай **ближе к РФ для скорости**: **Frankfurt**, **Amsterdam** или
   **Stockholm**. ⚠️ Регион потом не сменить — выбирай сразу правильный.
4. **Верификация картой:** нужна банковская карта. Oracle сделает временную блокировку
   ~$100 и **сразу вернёт** — реальных списаний за Always Free нет. (Если российская карта
   не проходит — подойдёт любая зарубежная/виртуальная.)
5. Дождись письма «Your account is ready».

---

## ЧАСТЬ 2. Создать бесплатный сервер (~5 мин)

1. Войди в консоль → меню (☰) → **Compute** → **Instances** → **Create instance**.
2. **Image and shape** → **Edit**:
   - **Image:** выбери **Canonical Ubuntu 22.04** (или 24.04).
   - **Shape:** **Change shape** → вкладка **Ampere** → **VM.Standard.A1.Flex**.
     Поставь **2 OCPU** и **12 GB** (этого с запасом; можно и 1/6, тоже хватит).
   - ⚠️ Если пишет **«Out of capacity»** — поменяй регион доступности или попробуй позже,
     либо возьми **VM.Standard.E2.1.Micro** (1 ядро, тоже Always Free) — для одного человека хватит.
3. **Add SSH keys** — выбери **Generate a key pair for me** и **СКАЧАЙ приватный ключ**
   (кнопка *Save private key*). Без него не зайдёшь на сервер.
4. Нажми **Create**. Через ~1 мин статус станет зелёным **RUNNING**.
   Запиши **Public IP address** (он на странице инстанса).

### Открыть порт 443 в облачном фаерволе Oracle

1. На странице инстанса кликни на имя подсети рядом с **Subnet**.
2. **Security Lists** → открой **Default Security List**.
3. **Add Ingress Rules**:
   - **Source CIDR:** `0.0.0.0/0`
   - **IP Protocol:** **TCP**
   - **Destination Port Range:** `443`
   - **Add Ingress Rules**.

---

## ЧАСТЬ 3. Запустить установщик (~2 мин)

Зайди на сервер по SSH (с компьютера). Пользователь у Ubuntu-образа Oracle — `ubuntu`.

**Windows:** используй приложение **Termius** или PowerShell.
**Mac/Linux:** терминал.

```bash
# подключение (подставь путь к скачанному ключу и свой IP)
ssh -i /путь/к/ssh-key.key ubuntu@ТВОЙ_PUBLIC_IP
```

Затем одной командой скачай и запусти установщик:

```bash
sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/pachaninm-lab/pachanin-demo/claude/xray-endpoint-happ-36Qc0/vpn/oracle/setup-reality.sh)"
```

Скрипт сам всё поставит и в конце напечатает:
- **готовую ссылку `vless://...`** для Happ,
- **QR-код** (можно отсканировать камерой в Happ),
- сохранит все данные в `/root/vpn-reality-info.txt`.

---

## ЧАСТЬ 4. Подключить Happ (iPhone и Samsung — одинаково)

1. Скопируй ссылку `vless://...` из вывода скрипта (пришли её себе в Telegram/заметки).
2. Открой **Happ** → **«+»** → **Добавить из буфера обмена** (или отсканируй QR).
3. Выбери сервер **MY-VPN-FOREVER** → подключись.
4. Проверь: открой yandex.ru / youtube.com — должно грузиться.

Готово. Один сервер, на всех устройствах, **навсегда**.

---

## Если что-то не работает

| Симптом | Решение |
|---|---|
| Happ подключается, но сайты не грузятся | Проверь, что в Oracle открыт Ingress на **443/TCP** (Часть 2). |
| `DNS_PROBE` / не находит сайты | В Happ в настройках профиля выстави DNS-стратегию **AsIs** (резолв на клиенте). |
| Через пару недель IP начал тормозить | РКН мог пометить IP. В консоли Oracle: останови инстанс → смени Public IP (бесплатно) → пропиши новый IP в ссылке. |
| Хочешь сменить сайт-маскировку | Перезапусти: `sudo DEST=dl.google.com bash setup-reality.sh` |

### Полезные команды на сервере
```bash
sudo systemctl status xray      # статус
sudo systemctl restart xray     # перезапуск
sudo journalctl -u xray -n 50   # логи
cat /root/vpn-reality-info.txt  # все данные и ссылка
```
