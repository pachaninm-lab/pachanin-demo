# EuServ — Бесплатный VPN навсегда (VLESS+Reality)

EuServ даёт бесплатный VPS в Германии навсегда. Карта не нужна.
Раз в месяц нажать кнопку Renew (бесплатно, просто подтверждение).

---

## Шаг 1. Заказать бесплатный VPS

1. Войди в панель: **https://www.euserv.com/en/** → Login
2. Перейди: **Virtual Servers → VS2-Free → Order**
3. Выбери **Debian 12** (или Ubuntu 22.04)
4. Нажми **Order** — сервер создастся за ~5 минут
5. На email придут данные: **IPv6-адрес** и **root-пароль**

Запиши IPv6-адрес — он выглядит как `2a02:xxxx:xxxx::1`

---

## Шаг 2. Подключиться по SSH

**На Windows** — скачай [Termius](https://termius.com) (бесплатный, поддерживает IPv6).

**На Mac/Linux** — терминал:

```bash
ssh root@ТВОЙIPv6
```

Пример:
```bash
ssh root@2a02:748:a800::1
```

При подключении введи root-пароль из письма.

---

## Шаг 3. Запустить установщик

После входа по SSH — одна команда:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/pachaninm-lab/pachanin-demo/claude/xray-endpoint-happ-36Qc0/vpn/euserv/setup-euserv.sh)
```

Скрипт сам:
- Установит Xray-core
- Сгенерирует ключи Reality
- Запустит VPN на порту 443
- Напечатает ссылку `vless://` для Happ

---

## Шаг 4. Добавить в Happ

1. Скопируй ссылку `vless://...` из вывода скрипта
2. В Happ: **«+»** → **Вставить из буфера** → подключись

---

## Раз в месяц (важно!)

EuServ бесплатный, но требует подтверждения раз в месяц:

1. Зайди на **https://www.euserv.com**
2. **My Services → VS2-Free → Renew Contract**
3. Нажми Renew — это бесплатно

EuServ пришлёт напоминание на email за несколько дней до истечения.

---

## Если что-то не работает

```bash
# Статус Xray
systemctl status xray

# Логи
journalctl -u xray -n 30

# Перезапуск
systemctl restart xray

# Твои данные и ссылка
cat /root/vpn-info.txt
```

---

## Почему IPv6

EuServ даёт только IPv6-адрес (без IPv4). Это работает если:
- Твой домашний интернет поддерживает IPv6 (большинство российских провайдеров: Ростелеком, Билайн, МТС, Мегафон)
- Проверь: открой **https://test-ipv6.com** — должно показать "10/10"

Если IPv6 нет — напиши, найдём другое решение.
