# Serv00 + VLESS-Reality — бесплатный VPN навсегда

Serv00 — польский бесплатный хостинг с реальным SSH-шеллом (FreeBSD).
Регистрация только по email, карта не нужна, бесплатно навсегда.

---

## Шаг 1. Регистрация на Serv00 (~3 мин)

1. Открой **https://www.serv00.com** → кнопка **"Create Free Account"**.
2. Заполни форму (имя, фамилия, email, страна — выбери любую).
   - Email должен быть настоящий — придёт письмо с данными для входа.
3. Жди письма (обычно ~1-5 минут). В письме будет:
   - Имя пользователя
   - Временный пароль
   - Имя сервера (например `s1.serv00.com` или `s2.serv00.com`)

> ⚠️ Если регистрация временно закрыта ("Registration is temporarily suspended") —
> попробуй снова через несколько часов. Serv00 периодически открывает слоты.

---

## Шаг 2. Включить SSH и найти порт (~2 мин)

1. Войди в панель: **https://panel.serv00.com** (логин/пароль из письма).
2. **SSH → Enable SSH access** — нажми кнопку чтобы активировать.
3. Запиши данные подключения: хост (`s1.serv00.com`) и SSH-порт
   (в письме или в разделе SSH — порт НЕ стандартный 22, обычно в районе 25128).
4. **Additional functions → Port Manager → TCP Ports** → запиши один из выделенных тебе портов.
   Это будет твой **VPN-порт** (например `12345`).

---

## Шаг 3. Подключиться по SSH

**На Windows** — установи [Termius](https://termius.com) (бесплатный).
**На Mac/Linux** — терминал.

```bash
ssh -p SSH_ПОРТ твой_логин@s1.serv00.com
```

(SSH_ПОРТ — из письма/панели, это НЕ 22)

---

## Шаг 4. Запустить установщик

После входа по SSH, одной командой:

```bash
curl -fsSL https://raw.githubusercontent.com/pachaninm-lab/pachanin-demo/claude/xray-endpoint-happ-36Qc0/vpn/serv00/setup-serv00.sh | sh -s VPN_ПОРТ
```

Где **VPN_ПОРТ** — порт из шага 2 (например `12345`).

Скрипт сам:
- Скачает sing-box для FreeBSD
- Сгенерирует UUID, ключи Reality
- Запустит VPN
- Напечатает ссылку для Happ

---

## Шаг 5. Добавить в автозапуск (важно!)

Чтобы VPN запускался автоматически после перезагрузки сервера:

1. В панели Serv00: **Additional functions → Run your own applications**
2. Добавь процесс:
   - **Command:** `/home/ТВОЙлогин/bin/sing-box run -c /home/ТВОЙлогин/singbox/config.json`
   - Нажми **Add** / **Save**

---

## Шаг 6. Добавить в Happ

1. Скопируй ссылку `vless://...` из вывода скрипта.
2. В Happ: **«+»** → **Вставить из буфера** → подключись.

---

## Если не работает

| Симптом | Решение |
|---|---|
| Сайты не открываются | Убедись что в настройках Serv00 порт разрешён (Port Manager → TCP). |
| sing-box упал | Зайди по SSH: `cat ~/singbox/sing-box.log` — посмотри ошибку. |
| IP поменялся | После перезагрузки Serv00 IP может меняться. Переделай ссылку: `cat ~/singbox/vpn-info.txt` покажет старый IP, узнай новый через `curl https://api.ipify.org`. |

### Полезные команды

```bash
# Статус
ps aux | grep sing-box

# Лог
tail -f ~/singbox/sing-box.log

# Перезапуск
pkill -f "sing-box run"; sleep 1
nohup ~/bin/sing-box run -c ~/singbox/config.json > ~/singbox/sing-box.log 2>&1 &

# Твои данные и ссылка
cat ~/singbox/vpn-info.txt
```

## Примечание

Serv00 — бесплатный, но на **shared hosting**. Это значит:
- Порт 443 недоступен (используем выделенный тебе порт)
- Reality на нестандартном порту всё равно намного лучше чем WS+TLS
- IP-адрес может меняться после редких перезагрузок сервера

Если Serv00 не доступен из твоей сети — напиши, найдём другое решение.
