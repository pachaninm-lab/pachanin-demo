# VPNjantit — Бесплатный VLESS+Reality для России

**Карта не нужна. Работает из России. Reality-протокол.**

VPNjantit — сервис с бесплатными серверами VLESS+Reality специально для России.
Аккаунт: только email + пароль. Срок: 7 дней, обновляется бесплатно.

---

## Шаг 1. Создать аккаунт (~2 мин)

1. Открой **https://www.vpnjantit.com/free-xray-vless-reality-russia**
2. Нажми **Create Account**
3. Введи email и пароль (без верификации карты)
4. Подтверди email — придёт письмо

---

## Шаг 2. Получить конфиг

1. Войди в личный кабинет
2. Выбери сервер **Russia** или любой другой (63 страны)
3. Нажми **Create Account** / **Generate Config**
4. Скопируй ссылку `vless://...`

---

## Шаг 3. Добавить в Happ

1. Скопируй ссылку `vless://...`
2. В Happ: **«+»** → **Вставить из буфера** → подключись

---

## Почему это работает

- Протокол **VLESS+Reality** маскируется под обычный HTTPS к microsoft.com
- РКН видит легитимный TLS — не блокирует
- Работает через домашний роутер и мобильный интернет

---

## Если кончился срок (7 дней)

Просто зайди снова: **https://www.vpnjantit.com/free-xray-vless-reality-russia**
→ Create Account → сгенерируй новый конфиг → замени в Happ.

Занимает ~2 минуты.

---

## Запасной вариант — подписка с авто-обновлением

Вместо VPNjantit можно добавить в Happ URL подписки —
конфиги обновляются автоматически каждые 2 часа:

```
https://raw.githubusercontent.com/igareck/vpn-configs-for-russia/main/BLACK_VLESS_RUS.txt
```

В Happ: **«+»** → **Добавить подписку** → вставь URL выше.
