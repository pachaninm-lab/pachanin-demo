# Прозрачная Цена — проектовый репозиторий

Этот репозиторий принят как основной проектовый Git-репозиторий для проекта «Прозрачная Цена».

## Назначение
Репозиторий используется как единая точка для:
- исходного кода платформы;
- hardening и integration-prep изменений;
- PR-процесса;
- технической документации по готовности, ограничениям и рискам;
- фиксации проверок build/test/readiness.

## Принятые правила
- main — только для проверенных изменений;
- рабочая ветка для текущего контура: `hardening14`;
- изменения по 14 проблемам, API/controllers/UI, readiness и test contour ведутся через PR;
- внешние live-интеграции не считаются закрытыми без отдельного подтверждения runtime/E2E.

## Текущий технический контур
По состоянию на текущий проход подготовлены и проверены:
- hardening-модули под 14 проблем;
- врезка в API/controllers/UI;
- локальный build/test contour уровня source/static readiness;
- отдельные отчёты по readiness и build/test contour.

## Что переносится следующим слоем
1. Структура монорепозитория проекта.
2. Domain-core и integration-hardening слой.
3. API/controllers/UI врезки.
4. Скрипты проверки readiness/build/test.
5. Техническая документация по ограничениям и статусу интеграций.

## Честная граница готовности
Состояние `code-ready` и `source-verified` не равно `live-integrated`.
Для полного подтверждения нужны:
- полноценные runtime-зависимости;
- поднятый API/Web;
- реальный dev/CI contour;
- внешние live-подключения (ЭДО, ГИС ЭПД, ГосЛог, реестры, банк).
