# IP Audit — platform-v7 / «Прозрачная Цена»

Дата: 2026-06-15. Цель: оценить чистоту прав на код/бренд для передачи
покупателю. Часть пунктов — owner-side (юридическое подтверждение вне кода).

## 1. Кодовая база

- Приватный монорепо (`private: true` во всех package.json), без публичной
  лицензии — корректно для проприетарного продукта.
- Доменная логика (`packages/domain-core`, `lib/platform-v7/**`) — оригинальная,
  предметно-специфичная (зерно/сделки/деньги/приёмка/споры), не похожа на
  скопированный сторонний продукт.
- Сторонний код используется как библиотеки через пакетный менеджер (см.
  `04-OSS_LICENSE_AUDIT.md`), не вкомпилирован копипастой (по результатам
  инвентаризации).

## 2. Бренд / товарные знаки

- **IP-001 (MEDIUM, owner-side):** «Прозрачная Цена» / «Pachanin» — статус
  регистрации товарного знака подтверждается владельцем (вне кода). Рекомендация:
  приложить свидетельство/заявку Роспатента к DD-пакету.
- Доменные имена/хостинг: Vercel (`pachanin-web.vercel.app`), Netlify
  (vermillion-kitsune / gleaming-mandazi) — права на основной домен подтверждает
  владелец.

## 3. Авторство и передача прав

- **IP-002 (MEDIUM, owner-side):** нет `CODEOWNERS`/CLA/договоров об отчуждении
  прав в репозитории. Для продажи нужны: трудовые/подрядные договоры с
  разработчиками с передачей исключительных прав (ст. 1295/1296/1288 ГК РФ),
  подтверждение, что AI-ассистированный код принадлежит владельцу.
- Git-история ведётся; авторство коммитов прослеживается.

## 4. Сторонние сервисы в коде (зависимости-сервисы)

- Аналитика/мониторинг в CSP allowlist: Yandex Metrica (`mc.yandex.ru`),
  Sentry (`*.sentry.io`). **IP-003 (LOW):** подтвердить владение этими
  аккаунтами/договорами при передаче.
- Внешние интеграции (банк/ФГИС/ЭДО/ЭПД/GPS/лаб/1С) — только shells за
  ENV+credentials, никаких встроенных проприетарных SDK партнёров (по
  инвентаризации) ⇒ нет «унаследованных» лицензионных обязательств партнёров до
  подключения.

## Чеклист передачи IP (owner-side)
- [ ] Договоры отчуждения исключительных прав на ПО от всех контрибьюторов.
- [ ] Свидетельства на товарный знак «Прозрачная Цена» (или статус заявки).
- [ ] Передача прав на домены/хостинг-аккаунты/аналитику.
- [ ] Подтверждение прав на контент (тексты/иконки/шрифт Inter — OFL).
- [ ] Реестр сторонних компонентов и их лицензий (см. OSS-аудит).

## Вердикт (IP)
Код оригинальный и проприетарный, сторонний код изолирован в зависимостях.
Юридическая «чистота» (отчуждение прав, товарный знак, домены) — owner-side и
закрывается документами, а не кодом.
