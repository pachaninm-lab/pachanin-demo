# Acceptance criteria

- Page `/platform-v7/market-entry` renders without changing shell or route constants.
- User can select a sourced price and route.
- Page calculates logistics cost per ton and delivered price.
- User can create a buy or sell intent in the page session.
- Launch gate shows price, party, logistics, counterparty, document and money readiness.
- Links point to existing lot, RFQ, market RFQ and bank surfaces.
- Unit test covers flow order, page rendering and intent creation.
