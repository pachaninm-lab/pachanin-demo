# Scope boundary

Market entry is a pre-deal surface.

It can:
- show sourced price records;
- calculate route-adjusted price;
- create a session-level buy or sell intent;
- show readiness before deal launch;
- link into existing execution surfaces.

It cannot:
- replace the execution runtime;
- move money;
- bypass documents;
- bypass counterparty checks;
- behave as an exchange;
- change role access rules.
