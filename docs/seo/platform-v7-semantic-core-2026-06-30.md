# SEO semantic core for platform-v7

Date: 2026-06-30
Status: controlled pilot / pre-integration
Scope: public platform-v7 contour only

## Positioning

Прозрачная Цена is not a generic grain marketplace. The SEO position is: digital execution contour for a grain deal after the price is agreed.

Core formula:

price and admission -> deal -> logistics -> acceptance -> quality -> documents -> payment basis -> dispute -> evidence.

## Primary semantic cluster

Target intent: the user is not just looking for a counterparty or price. The user wants control over execution, documents, payment basis and dispute evidence.

Target phrases:

- цифровой контур исполнения зерновой сделки
- исполнение зерновой сделки после согласования цены
- контроль исполнения зерновой сделки
- зерновая сделка рейс приёмка документы расчёт
- основание для оплаты зерновой сделки
- доказательства исполнения зерновой сделки

Main public URL:

- /platform-v7

CTA:

- demo deal
- question form
- organization connection

## Negative or low-priority traffic

These phrases are not primary SEO targets because they attract broad marketplace traffic and may reduce lead quality:

- купить зерно
- продать зерно
- цены на пшеницу
- доска объявлений зерно
- зерно оптом
- агро маркетплейс
- бесплатные объявления зерно

Use only if a future public page clearly connects the query to execution control after price, not to generic trading.
